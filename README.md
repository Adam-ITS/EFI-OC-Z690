# EFI-OC-1.0.5-Z690

EFI folder for GigaByte Z690 UD4 motherboard
Works for MacOS 15.7.1 Sequoia with hardware config :
- i5-12400F
- RX6600 8G
- 64Go DDR4
- NVMe WD Black
